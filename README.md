# Day Trader Pro

Personal-use day-trading **monitoring/screening** dashboard for US and Canadian stocks.
Runs entirely on your machine. It never places trades and nothing it shows is investment advice.

> Work in progress — full setup docs land with milestone 5. Quick start:
>
> ```
> npm install
> npm run dev
> ```
>
> Backend on http://localhost:4400, UI on http://localhost:5173.
> With no API keys configured, the app runs in clearly-labeled SIMULATED mode.
